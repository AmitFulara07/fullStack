# RBACS (Role-Based Academic Collaboration System)

A web platform that enables structured tracking of student project progress while facilitating continuous mentorship and academic guidance within a college ecosystem.

## Setup

1. Rename `server/.env.example` to `server/.env` and update the environment variables.
2. In the `server` directory, run `npm install` and then `npm run dev` to start the backend.
3. In the `client` directory, run `npm install` and then `npm run dev` to start the frontend.
